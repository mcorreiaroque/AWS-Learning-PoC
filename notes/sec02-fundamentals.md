**What is a server composed of?**

To understand the Cloud, we must first understand the physical building blocks of a server. A server is essentially a powerful computer composed of five core components

    -> Compute: The "Brain" of the server.
    It performs all the mathematical calculations and executes the instructions of your code.
    -> Memory: The "Short-term memory." It provides extremely fast access to data that the CPU is currently using. More RAM allows the server to handle more simultaneous tasks or users.
    -> Storage: The "Long-term memory." This is where your files, operating system, and application code are permanently stored
    -> Database: A specialized way to store data in a structured and organized format. While storage holds files, a database holds searchable data
    -> Network: The "Communication system." It consists of routers, switches, and DNS servers that allow the server to talk to the internet and to other servers.
